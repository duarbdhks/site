🎯 구성 의도

이 로드맵은 ClickHouse를 단순히 SQL 문법으로 배우는 자료가 아니라, `CDC → Kafka → Flink → ClickHouse → API` 통계 파이프라인을 운영하는 관점에서 읽도록 구성했습니다. 각 주차는 과제형이 아니라 강의 노트형입니다. 개념을 먼저 설명하고, 쉬운 비유와 실무 사례를 붙인 뒤, 마지막에 판단 기준과 용어를 정리합니다.

기존 버전보다 각 주차 분량을 늘려 10분 이상 학습할 수 있도록 구성했습니다. 코드 예시는 최소화했고, 대신 “왜 이 개념이 필요한지”, “어떤 상황에서 문제가 되는지”, “실무에서 어떤 선택을 해야 하는지”를 중심으로 작성했습니다.
